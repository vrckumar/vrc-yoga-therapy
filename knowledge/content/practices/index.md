# Practice Index

This page is the canonical lookup surface for practice names used across the knowledge library.
It is meant to help readers move from a Sanskrit term, English label, alias, or transliteration
to the correct content page quickly.

## How To Use This Index

- If a term appears in an ailment or practice page, the first meaningful mention should link to
  its canonical page.
- When two names refer to the same concept, this index points to the canonical page rather than
  duplicating the entry.
- Some entries are foundational reference pages used mainly as support targets for other content.
